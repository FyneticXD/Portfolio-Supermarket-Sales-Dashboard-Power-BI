Supermarket Sales Dashboard - Power BI
📊 Deskripsi Proyek
Proyek ini merupakan portofolio analisis data dan visualisasi interaktif menggunakan Power BI. Dashboard ini dirancang untuk menganalisis performa penjualan historis dari sebuah jaringan supermarket yang memiliki tiga cabang utama. Tujuan utama dari proyek ini adalah untuk menggali wawasan (insights) terkait tren pendapatan, demografi pelanggan, dan performa setiap lini produk untuk mendukung pengambilan keputusan.
📂 Dataset
Dataset yang digunakan (`supermarket\_sales.csv`) berisi data transaksi penjualan pada awal tahun 2019.
Fitur utama dalam dataset ini meliputi:
Lokasi & Cabang: Cabang (A, B, C) dan Kota (Yangon, Naypyitaw, Mandalay).
Demografi Pelanggan: Tipe Pelanggan (Member / Normal) dan Jenis Kelamin.
Detail Transaksi: Lini Produk (Health and beauty, Electronic accessories, dll), Harga Satuan, Kuantitas, Metode Pembayaran (Cash, Credit card, Ewallet), dan COGS (Cost of Goods Sold).
Kepuasan Pelanggan: Penilaian (Rating) dari setiap transaksi (skala 1-10).
💡 Fitur Dashboard & Wawasan Utama
Dashboard Power BI (`Supermarket Sales Dashboard Power BI.pbix`) mencakup analisis berikut:
Ringkasan Pendapatan (Revenue Overview): Menampilkan total pendapatan dan profitabilitas berdasarkan cabang dan lini produk.
Analisis Pelanggan: Perbandingan perilaku pembelian antara pelanggan Member dan Normal, serta distribusi berdasarkan jenis kelamin.
Metode Pembayaran: Tren penggunaan E-wallet, Cash, dan Credit Card.
Performa Lini Produk: Mengidentifikasi kategori produk dengan penjualan tertinggi dan rating kepuasan pelanggan terbaik.
🛠️ Teknologi yang Digunakan
Alat Bantu Utama: Power BI Desktop
Pemrosesan Data: Power Query, DAX (Data Analysis Expressions)
Sumber Data: File CSV
🚀 Cara Menjalankan Proyek
Pastikan Anda telah menginstal Power BI Desktop di perangkat Anda.
Clone repositori ini atau unduh semua filenya ke direktori lokal.
Buka file `Supermarket Sales Dashboard Power BI.pbix`.
Jika terdapat peringatan pembaruan data, buka pengaturan sumber data (Data Source Settings) dan arahkan path ke file `supermarket\_sales.csv` yang baru saja Anda unduh.
Klik Refresh untuk memuat ulang data.
